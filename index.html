Catalyst

Catalyst aims to simplify the generation of unit tests for React framework with an emphasis on enzyme

Landing Page

With a simple click of a button, you can choose your project of choice and its content will be displayed

How to use

Using our interactive GUI, you'll be able to easily choose which file you want to create tests and create tests while viewing your code side by side. Afterwards, click on generate tests and this code will be displayed for reviewing.

Create and Export Functional Test files

Test files can be previewed and exported to a test folder in the directory of the chosen project
